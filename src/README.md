**Rough Plan**

1. From all investor texts extract “decisions” or “actions”.

2. For each 'action', create a mini descriptive document with:

   **METADATA:** Date, Share, Industry etc.
   **CONTEXT:** The situation in front of investors
   **ACTION:** The action taken
   **REASON:** Reasoning for that action.
